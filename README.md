# Italiano

Questo repository contiene materiali da usare come fonte per rispondere a domande tramite AI.

## Scopo del repository

Lo scopo di questo repository è raccogliere file, documenti e materiali utili affinché l'AI possa consultarli e rispondere basandosi esclusivamente o principalmente sui contenuti presenti qui.

## Come usare questo repository

Quando analizzi questo repository:

1. Leggi prima questo README.
2. Usa i file del repository come fonte principale.
3. Se una risposta riguarda un documento specifico, fai riferimento al file da cui hai preso l'informazione.
4. Non inventare informazioni non presenti nei file.
5. Se un'informazione non è presente, dillo chiaramente.
6. Dai priorità ai file più aggiornati o più completi.

## Struttura del repository

- `/documenti/`  
  Contiene i documenti principali da consultare.

- `/appunti/`  
  Contiene appunti, riassunti e materiali di studio.

- `/fonti/`  
  Contiene eventuali testi originali, PDF, fonti o materiali di riferimento.

- `/immagini/`  
  Contiene immagini, screenshot o scansioni utili.

## Ordine consigliato di lettura

Per rispondere alle domande, l'AI dovrebbe consultare i materiali in questo ordine:

1. File di sintesi o riassunto generale.
2. Documenti principali.
3. Appunti secondari.
4. Immagini o screenshot, solo se necessari.

## Regole per l'AI

L'AI deve:

- basarsi sui contenuti presenti nel repository;
- distinguere tra informazioni certe e interpretazioni;
- citare o indicare il file usato quando possibile;
- segnalare eventuali contraddizioni tra i file;
- non usare informazioni esterne, salvo richiesta esplicita;
- rispondere in modo chiaro, ordinato e utile allo studio.

## Note sui file

I file possono contenere materiali scolastici, appunti, riassunti, immagini o documenti da analizzare. Se alcuni file risultano incompleti o poco leggibili, l'AI deve segnalarlo invece di dedurre informazioni non sicure.
